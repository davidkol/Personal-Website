VR Spell Fighter - Github

I started this project to try working with unique hardware and networks. This project was in Unreal Engine, using the HTC Vive.

Using all new technology and software meant I was locked into using the engines custom visual Blueprints language, rather than C++ code. However using Blueprints was an educational exercise, as it is a direct visual representation of C++, necessitating better code style and organization.

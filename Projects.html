Magic Shotgun - Github

The objective of this project was to improve my ability to teach myself new skills and manage my time. I placed a hard limit of 3 months to get a working, playable game, and to learn as much as possible about these key areas:

  • Mechanics (C++)
  • Animations (Blender)
  • A.I. (C++/Blueprints)

VR Spell Fighter - Github

I started this project to try working with unique hardware and networks. This project was in Unreal Engine, using the HTC Vive. Starting from scratch made it difficult as I had to figure out how to build many of the basic elements from nothing, leaving little time to build the online element of the game. The entire project took 2 months in the summer of 2016. Hopefully when the next wave of VR hardware is released I can continue where I left off.
